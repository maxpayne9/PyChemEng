Loading PyChemEng

Most of PyChemEng is accessed through a single module, chemeng. 

from chemeng import *

Elemental database

Masses of components are calculated using the elemental database within PyChemEng. This contains accurate isotopic masses and abundance data: 

print elements['Ca']
#Element{Ca, Z=20, AW=40.078, 24 isotopes}
print elements['Li'].mass
#6.941

The average atomic weight (AW) is calculated from the abundance of each isotope and its mass. We can examine this information by accessing the isotope using its neutron number, e.g. 

print elements['C'].isotopes[6]
#Isotope{C, Z=6, N=6, M=12.0(0.0), P=0.9893}

Or by directly specifying the proton (Z) and neutron (N) number as a tuple 

print elements[(1,2)] #Tritium
#Element{T, Z=1, AW=3.0160492777}

Molar/Mass balances

The first task of a chemical engineer is to perform mass/molar balances. PyChemEng lets you build molar balances very easily using the Components class. To create it, we provide it with a dictionary of chemical formulas and their molar abundances. For example: 

air = Components({'O2':0.21, 'N2':0.79})
print air
#C{'N2':0.79, 'O2':0.21}

You can add/mix components and scale them to make new sets of component.

For example, here is a calculation of a stochiometic methane fuel mix, its mole fractions, and its mass flux 

fuel = Components({'CH4':1.0})
stochiometricMix = fuel + air * (3.0 * fuel['CH4'] / air['O2'])
print stochiometricMix
#C{'CH4':1, 'N2':11.2857, 'O2':3}
print stochiometricMix.normalised()
#C{'CH4':0.0654206, 'N2':0.738318, 'O2':0.196262}
print stochiometricMix.total()
#15.2857142857
print stochiometricMix.totalMass()
#428.190088571

We can also break the stream down to its elemental composition. 

print stochiometricMix.elementalComposition()
#C{'C':1, 'H':4, 'N':22.5714, 'O':6}
